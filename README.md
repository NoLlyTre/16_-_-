# Личный кабинет студента

Проект информационной системы «Личный кабинет студента», предназначенной для
взаимодействия студентов и преподавателей.

## Основной функционал:
- просмотр расписания занятий;
- просмотр оценок и успеваемости;
- загрузка и сдача заданий;
- обмен сообщениями между студентом и преподавателем.
